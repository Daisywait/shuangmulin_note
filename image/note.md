### 1. 通过 OMEN Gaming Hub 切换（最简单）

1. 在 Windows 桌面按下 **Win 键**，搜索并打开 **OMEN Gaming Hub**（图标是一个彩色方块钻石）。
    
2. 在左侧栏点击你的电脑型号（例如 **OMEN 15** 或 **OMEN 17**）进入设置页面。
    
3. 在顶部的选项卡中找到 **“显卡切换器” (Graphics Switcher)**。
    
4. 你会看到两个选项：
    
    - **混合 (Hybrid)**：默认选项，省电但有延迟。
        
    - **独立显卡 (Discrete)**：**请勾选这一项**。
        
5. 点击“重新启动”。电脑重启后，核显将被禁用，屏幕将直接由 RTX 5060 驱动。
    

---

### 2. 如果在 Windows 里找不到该选项（通过 BIOS 切换）

如果在软件里没看到开关，可以直接在底层设置：

1. **重启电脑**，在屏幕亮起的一瞬间，反复狂点键盘上的 **F10** 键。
    
2. 进入 BIOS 界面后，使用方向键找到 **Configuration**（配置）菜单。
    
3. 寻找 **Graphics Switcher** 或 **Video Mode** 选项。
    
4. 将其值从 **Hybrid** 改为 **Discrete**（独立显卡）。
    
5. 按下 **F10** 保存并退出。
