---
类型: 命令
标签: [Linux, 文件查看]
---

- 命令
```bash
# 查看文件开头/末尾
head -n 1 data.csv

tail -n 50 app.log
```

```bash
# 持续查看日志

tail -f app.log
```

```bash
# 查看文件内容
cat file.txt

less file.txt
```

```bash
# 统计行数
wc -l file.txt
```

- 参数说明
```bash
# -n N: 只显示前/后 N 行
# -f: 持续跟踪输出
# -l: 统计行数
```

- 解释：快速浏览文件内容与日志。
