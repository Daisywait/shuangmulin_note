---
类型: 命令
标签: [Linux, 文件操作]
---

- 命令
```bash
# 查看当前目录
pwd
```

```bash
# 列出文件（含隐藏文件，长格式）
ls -la
```

```bash
# 切换目录
cd /path/to/dir

cd ..

cd ~
```

```bash
# 创建目录
mkdir mydir

# 创建嵌套目录
mkdir -p dir/subdir
```

```bash
# 创建文件
touch file.txt
```

```bash
# 查看文件内容
cat file.txt
```

```bash
# 复制文件/目录
cp src.txt dst.txt

cp -r dir1 dir2
```

```bash
# 移动/重命名
mv old.txt new.txt

mv file.txt /path/to/dir/
```

```bash
# 删除文件/目录（谨慎）
rm file.txt

rm -r dir

rm -rf dir
```

- 参数说明
```bash
# -a: 显示隐藏项
# -l: 长格式
# -r: 递归
# -f: 强制
# mkdir -p: 递归创建目录
```

- 解释：常用文件与目录操作汇总。
