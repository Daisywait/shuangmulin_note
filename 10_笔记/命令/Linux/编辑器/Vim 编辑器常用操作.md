- 命令
```bash
# 打开/创建文件（进入 Vim）
vi filename

vim filename
```

```bash
# 只读模式打开
vi -R filename

view filename
```

```bash
# Vim 内部常用命令
:q

:q!

:w

:wq

ZZ

:w newfile
```

- 参数说明
```bash
# :q    正常退出
# :q!   放弃修改并强制退出
# :w    保存但不退出
# :wq   保存并退出
# ZZ    保存并退出（大写）
# :w newfile  另存为新文件
```
- 解释：快速记住 Vim 基本打开/保存/退出与常用编辑流程。
