来源[[环境]]]

- 命令
```bash
# 把 HuggingFace 缓存指向数据盘。
echo "export HF_HOME=/data/linmiaoju/.cache/huggingface" >> ~/.bashrc
```

```bash
# 把 Conda 环境与包缓存指向数据盘。
echo "export CONDA_ENVS_PATH=/data/linmiaoju/.conda/envs" >> ~/.bashrc
echo "export CONDA_PKGS_DIRS=/data/linmiaoju/.conda/pkgs" >> ~/.bashrc
```

```bash
# 把 Torch 与通用缓存指向数据盘。
echo "export TORCH_HOME=/data/linmiaoju/.cache/torch" >> ~/.bashrc
echo "export XDG_CACHE_HOME=/data/linmiaoju/.cache" >> ~/.bashrc
```

```bash
# 重新加载环境变量。
source ~/.bashrc
```

- 参数说明
```bash
# HF_HOME/CONDA/TORCH: 指向数据盘缓存
```
- 解释：把大数据放在 / 会占满系统盘。
