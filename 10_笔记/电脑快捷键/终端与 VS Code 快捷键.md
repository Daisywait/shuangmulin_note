- 一句话：整理终端与 VS Code 的常用快捷键，减少鼠标操作。

- 命令
```bash
# 终端中打开 VS Code（当前目录）
code .
```

- 说明
  - 终端快捷键：
    - 打开终端：`Ctrl + Alt + T`
    - 复制：`Ctrl + Shift + C`
    - 粘贴：`Ctrl + Shift + V`
  - VS Code 快捷键：
    - 打开集成终端：`Ctrl + ~`
    - 分屏终端：`Ctrl + Shift + 5`

- 解释：将高频操作固化成快捷键组合，提高开发效率。
