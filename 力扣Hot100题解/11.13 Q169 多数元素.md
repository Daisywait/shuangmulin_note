#### 题目
给定一个大小为 `n` 的数组 `nums` ，返回其中的多数元素。多数元素是指在数组中出现次数 **大于** `⌊ n/2 ⌋` 的元素。
你可以假设数组是非空的，并且给定的数组总是存在多数元素。


#### 解法
Boyer–Moore 投票算法
```python
class Solution:
    def majorityElement(self, nums):
        candidate = None
        count = 0
        for num in nums:
            if count == 0:
                candidate = num
            count += (1 if num == candidate else -1)
        return candidate
```
把数组想成一场“投票”，多数元素的票数一定能压倒其他人。  
最终剩下的候选人就是多数元素。  
时间 O(n)，空间 O(1)。
