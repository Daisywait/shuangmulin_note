#### 题目
给你单链表的头节点 `head` ，请你反转链表，并返回反转后的链表。


#### 解法
###### 递归
```python
class Solution(object):
    def reverseList(self, head):
        if not head or not head.next:
            return head
        new_head=self.reverseList(head.next)
        head.next.next=head
        head.next=None
        return new_head
```
>new_head一直指着原始链表的最后一个元素，时间复杂度是0（N）

###### 迭代
```python
class Solution(object):
    def reverseList(self, head):
        prev=None
        curr=head
        while curr:
            next_node=curr.next
            curr.next=prev#感觉比斐波那契多了一步这个将当前节点指向前一个节点
            prev=curr
            curr=next_node
        return prev
```
>时间复杂度是0（N）空间是O（1）
