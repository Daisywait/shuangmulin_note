#### 题目：
假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。
每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？

#### 解法
可以使用斐波那契数列，本质上就是斐波那契数列，但是斐波那契需要用到递归调用函数，当n变大的时候太费时了，所以变成使用下面的动态规划也叫迭代。时间 `O(n)`，空间 `O(1)`
```python
class Solution(object):
    def climbStairs(self, n):
        if n==1:
            return 1
        a,b=1,2
        for _ in range (3,n+1):
            a,b = b , a+b
        return b
```
