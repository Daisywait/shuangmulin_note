#### 题目
给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。
你只能选择 某一天买入这只股票，并选择在 未来的某一个不同的日子卖出该股票。设计一个算法来计算你所能获取的最大利润。返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

#### 解法
>！千万不要用循环
>想保证买入在前，卖出在后只要从索引为0开始去比较就好了！！！
>我还想着用删除元素、maxmin函数、enumerate。。。

用动态规划
“Dynamic” 不是“动态变化”的意思，而是指**分阶段决策**；  
“Programming” 不是“写代码”，而是“规划”。  
直译其实应该是 “分阶段最优决策”。

```python
class Solution(object):
    def maxProfit(self, prices):
        if not prices:
            return 0
        min_price=float('inf')
        max_profit=0
        for price in prices:
            if price<min_price:
                min_price=price
            elif price-min_price>max_profit:
                max_profit=price-min_price
        return max_profit
```
