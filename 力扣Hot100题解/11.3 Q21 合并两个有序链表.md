#### 题目描述
将两个升序链表合并为一个新的 升序 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

#### 解法
这里就需要补充一些[[链表]]的知识了
```python
class Solution(object):
    def mergeTwoLists(self, list1, list2):
        dummy=ListNode(-1)#创建一个假头，为了方便往里面加节点
        tail=dummy#将tail指向dummy的第一个假头也就是-1的位置
        while list1 and list2:
            if list1.val<=list2.val:#链表用ListNode.val获取当前节点的值，默认刚开始处于第一个节点
                tail.next=list1#将tail指向list1的第一个节点，但这时本身节点1自己连着下一个节点，等到下次指向新的就会替换掉原来的
                list1=list1.next#移动到下一个节点
            else:
                tail.next=list2
                list2=list2.next
            tail=tail.next
        tail.next=list1 if list1 else list2#将剩余的接到
        return dummy.next
```
*dummy.next 永远指向结果链表的开始，tail.next 只负责挂接新的节点，不影响 dummy.next 的完整性。*
