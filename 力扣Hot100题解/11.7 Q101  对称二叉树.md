#### 题目
给你一个二叉树的根节点 `root` ， 检查它是否轴对称。


#### 解法
用递归，时间复杂度是O（n)
>这个是通过**系统调用栈**来保存函数递归状态的。
>- 如果树是**完全平衡的**，递归深度 ≈ 树高 `h = log₂(n)`；
>- 如果树是**极度不平衡的（像链表）**，递归深度 = 节点总数 `n`。空间复杂度：O(h)（最坏 O(n)）。

```python
# Definition for a binary tree node.
class Solution(object):
    def isSymmetric(self, root):
        def isMirror(l,r):
            if not l and not r:
                return True
            if not l or not r:
                return False
            if l.val!=r.val:
                return False
            return isMirror(l.left,r.right) and isMirror(l.right,r.left)
        if not root:
            return True
        return isMirror(root.left,root.right)
```
