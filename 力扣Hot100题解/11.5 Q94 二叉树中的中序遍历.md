#### 题目描述
给定一个二叉树的根节点 `root` ，返回 _它的中序遍历_ 。
#### 解法
中序遍历指：左->根->右
>_*补充*_
>前序遍历：中 -> 左 -> 右
>后序遍历：左 -> 右 -> 中
###### 递归
```python
# class TreeNode(object):
class Solution(object):
    def inorderTraversal(self, root):
        res=[]
        def inorder(node):
            if node is None:
                return
            inorder(node.left)#再一次调用的时候应该是子节点的left
            res.append(node.val)
            inorder(node.right)
        inorder(root)
        return res
```
###### 颜色标记法
```python
    def inorderTraversal(self, root):
        WHITE, GRAY = 0, 1
        res = []
        stack = [(WHITE, root)]
        while stack:
            color, node = stack.pop()
            if not node:
                continue
            if color == WHITE:
                # 中序遍历：左 -> 中 -> 右
                stack.append((WHITE, node.right))
                stack.append((GRAY, node))
                stack.append((WHITE, node.left))
            else:
                res.append(node.val)
        return res
```
