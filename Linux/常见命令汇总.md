#### 文件查看

```bash
head -n 1 data.csv
```

`head` ：意思是“取文件的开头部分”。默认情况下它会显示文件的前 10 行。  
`-n 1` ： `head` 只显示 *前 1 行*。  
    `-n` 是 “number of lines” 的意思。  
    如果你写 `-n 5`，那就会显示前 5 行。
`data.csv` ：要查看的文件名。

### **APT** (Debian/Ubuntu 等)

bash

# 卸载软件包但保留配置文件
sudo apt remove 软件包名

# 彻底卸载（包括配置文件）
sudo apt purge 软件包名

# 示例：彻底卸载 Firefox
sudo apt purge firefox
