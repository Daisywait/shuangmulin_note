# LMJ_NOTE 笔记仓库

这是一个以 Obsidian 为核心的个人知识与项目记录库，强调“随记 → 卡片 → 参考资料 → 项目推进”的流转。

## 目录结构

- [00_随记](00_随记/README.md)：未处理、杂记、临时记录。
- [10_笔记](10_笔记/README.md)：可复用的卡片化知识。
- [20_参考资料](20_参考资料/README.md)：论文/文档摘录与来源材料。
- [30_项目进度和维护](30_项目进度和维护/README.md)：项目推进与记录。

## 使用约定（简要）

- 新内容先放在 00_随记，定期整理并移动到对应目录。
- 可复用知识沉淀到 10_笔记；完整来源材料放到 20_参考资料。
- 项目过程、决策与日志记录放到 30_项目进度和维护。
