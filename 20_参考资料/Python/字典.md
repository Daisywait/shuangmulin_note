#### 创建字典
方法一：使用`{}`字面量语法
```python
	xinhua = {
    '麓': '山脚下',
    '路': '道，往来通行的地方；方面，地区：南～货，外～货；种类：他俩是一～人',
    '蕗': '甘草的别名',
    '潞': '潞水，水名，即今山西省的浊漳河；潞江，水名，即云南省的怒江' }
	```
方法二：使用内置函数`dict`、`zip`或者是字典的生成式语法
```python
	person = dict(name='王大锤', age=55, height=168, weight=60, addr='成都市武侯区科华北路62号1栋101')
	items1 = dict(zip('ABCDE', '12345'))
	print(items1)  # {'A': '1', 'B': '2', 'C': '3', 'D': '4', 'E': '5'}
	items3 = {x: x ** 3 for x in range(1, 6)}
	```
	
#### 字典中的键类型
必须是[[hashable]]

##### 获取字典长度
使用`len`函数；
```python
	# 创建一个字典
	person = {
    "name": "Alice",
    "age": 25,
    "city": "Beijing",
    "job": "Engineer"}

	# 获取字典长度（键值对的数量）
	length = len(person)
	print(f"字典长度: {length}")  # 输出: 字典长度: 4
	```
	
#### 查找键值
`in` 操作符在字典中默认查找的是键(key)

#### 字典方法
`values` 是字典 `bracket_map` 的一个方法，因此应该在 `values` 后加上括号
```python
if char in bracket_map.values():
```
