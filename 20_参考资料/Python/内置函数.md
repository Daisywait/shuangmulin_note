### enumerate
###### 基本语法：

```python
enumerate(iterable, start=0)
```
- `iterable`: 任何可迭代对象（列表、字符串、元组等）
- `start`: 索引起始值，默认为 0

###### 用法：

>在循环时同时获取**索引**和**元素值**。

```python
fruits = ['apple', 'banana', 'cherry']

# 不使用 enumerate
for i in range(len(fruits)):
    print(f"索引 {i}: {fruits[i]}")

# 使用 enumerate（更简洁）
for i, fruit in enumerate(fruits):
    print(f"索引 {i}: {fruit}")

# 输出：
# 索引 0: apple
# 索引 1: banana  
# 索引 2: cherry
``
