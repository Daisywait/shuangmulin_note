# 一、数据读取
好，我们来拆解这条命令：

```bash
!head -n 1 data.csv
```

这是在 **Colab（或 Linux 命令行）** 中执行的命令，其中：

- `!` ：告诉 Colab「接下来这一行是系统命令（Shell 命令）」，不是 Python 代码。  
    所以你在 Colab 里写 `!ls`、`!pwd`、`!head` 这种，都是在调用底层 Linux 命令。
    
- `head` ：意思是“取文件的开头部分”。默认情况下它会显示文件的前 10 行。  
    这个命令在 Linux 和 macOS 系统里都非常常用。
    
- `-n 1` ：这个选项告诉 `head` 只显示 **前 1 行**。  
    `-n` 是 “number of lines” 的意思。  
    如果你写 `-n 5`，那就会显示前 5 行。
    
- `data.csv` ：是你要查看的文件名。
    

综合起来的意思就是：

> “显示文件 `data.csv` 的第一行内容。”

而因为 CSV 文件的第一行通常是列名，所以这条命令恰好能快速显示出所有的列标题。

---

小补充：  
如果你想看最后一行而不是第一行，对应命令是

```bash
!tail -n 1 data.csv
```

`tail` 就是“文件尾部”的意思。

要不要我给你对比一下 `head`、`tail` 和 `cat` 三个命令的区别？这三个组合起来在看 CSV 时非常好用。
